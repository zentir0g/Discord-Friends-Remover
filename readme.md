# Discord Friends Remover 🗑️

A powerful tool to manage and remove Discord friends in bulk with thread support for faster processing. Created by **Zentirog**.

---

## 📌 Features

- **List all your Discord friends** - View your entire friends list
- **Blacklist system** - Select friends you want to keep
- **Multi-threaded removal** - Fast processing with configurable threads (4-10 recommended)
- **Simple CLI interface** - Easy to use command-line tool

## ⚙️ How to Use

### Installation
1. **Download** the `index.py` file
2. Make sure you have **Python 3.x** installed

### Usage Steps
1. **Run the program**
   ```bash
   python index.py
   ```

2. **Enter your Discord token** when prompted
   > ⚠️ Keep your token secure! Never share it with anyone

3. **Review your friends list** - The program will display all your friends

4. **Blacklist friends** - Input the usernames or IDs of friends you want to keep (they won't be removed)

5. **Configure threads** - Set between **4-10 threads** for optimal performance
   - 4-6 threads: Slower but more stable
   - 7-10 threads: Faster removal but more resource-intensive
   - Recommended: Start with 6 threads

6. **Start the removal** - The program will begin unfriending all non-blacklisted users
